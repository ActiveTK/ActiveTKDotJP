
# ActiveTK.jp

## 本プロジェクトについて
本プロジェクトは、[ActiveTK.jp](https://www.activetk.jp/)のソースコードです。
本プロジェクトは、簡単に、安全で、無料で使える便利なWebツールを提供する事を目標としています。
サイト内の全てのツールは、特に指定が無い場合、全て無料でご利用いただけます。
